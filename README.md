# Intelligent Nutritional System

A Python-based application that generates and dynamically adjusts personalized diet plans based on user data and progress.

## Overview

This system simulates an adaptive nutritional assistant. Instead of using static diet plans, it continuously updates recommendations based on user feedback and performance.

## Features

- Personalized diet generation  
- Progress tracking  
- Automatic diet adjustment  
- Integration of user observations and symptoms  
- Classification-based decision model  
- Graphical user interface (CustomTkinter)  
- Optional AI-based diet generation  

## How It Works

1. The user provides personal data  
2. The system generates an initial diet  
3. The user records progress  
4. The system analyzes the data  
5. The diet is maintained, adjusted, or changed  

## Technologies

- Python  
- CustomTkinter  
- Scikit-learn  
- OpenAI API (optional)
